## Installation

### npm

```sh
npm install
```

### yarn

```sh
yarn install
```

## Run App

### Development Mode

### npm

```sh
npm run dev
```

### yarn

```sh
yarn dev
```

### Produdction Mode

### npm

```sh
npm run preview
```

### yarn

```sh
yarn preview
```

## Build App

### npm

```sh
npm run build
```

### yarn

```sh
yarn build
```
